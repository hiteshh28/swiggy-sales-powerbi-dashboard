# Swiggy Sales Dashboard | Power BI

## Project Overview
This project presents an interactive Swiggy Sales Dashboard built using Power BI to analyze food delivery sales performance and customer ordering trends.

## Tools Used
- Power BI
- Data Visualization
- Data Analysis

## Key Features
- KPI cards displaying Total Orders, Total Sales, and Revenue
- Category-wise analysis of food items
- Interactive filters for dynamic insights
- Visualizations using bar charts, pie charts, and trend charts
